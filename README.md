## Installation

```
git clone https://github.com/redc86/slash
cd slash
pip install -r requirements.txt
python slash.py help
```

## Usage & Syntax

* Username Syntax : **python slash.py username**
* Mail Adress Syntax : **python slash.py mail_adress**

* Example : 
```
python slash.py redc86
```
